# EcommerceSQLProject

## Project Overview
This is a SQL-based E-commerce database system designed to manage products, customers, orders, and inventory. The project demonstrates database design, data manipulation, and analytical query skills.

## Tables Included
- **Customers**: Stores customer details (Name, Email, Phone, Address)
- **Products**: Product details including name, category, price, and stock quantity
- **Categories**: Product categories for organization
- **Suppliers**: Supplier information
- **Inventory**: Tracks product quantities supplied by suppliers
- **Orders**: Records customer orders with total amounts
- **OrderItems**: Stores individual products in each order

## Sample Queries
- List all orders with customer names and total amounts  
- Top 5 best-selling products  
- Customers with orders over $100  
- Products low in stock (<20)  
- Monthly revenue summary

## Skills Demonstrated
- SQL database design and normalization  
- Writing complex queries and joins  
- Using primary and foreign keys effectively  
- Stored proc
